# 무엇?

코드의 양을 줄이기 위해서 나온 if, else의 줄임버전

# 형식


```python
변수 = 표현식1 if 조건1 else 표현식2
```


```python
a = 10
b = 20
result = (a-b) if a == b else (a+b)
print(result)
```

    30
    

result 값은 a와 b가 같다면 a-b를 다르다면 a+b를 가지는 변수이다.  
따라서 a와b는 다르므로 10+20 즉 30이 출력되었다.
